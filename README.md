
# Registration From | Webpage using Bootstrap

This project implements a user authentication system with a responsive and user-friendly sign-in and sign-up form. Built using a combination of JavaScript for dynamic functionality, CSS for styling, HTML for structure, and Bootstrap for enhanced UI components.
